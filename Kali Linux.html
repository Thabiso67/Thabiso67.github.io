Kali Linux Virtual Machine

A Kali Linux virtual machine is a virtualised installation of the Kali Linux operating system that runs inside a host computer using virtualisation software such as VirtualBox. It allows Kali Linux to be used in an isolated and safe environment to practise cybersecurity and Linux tasks without affecting the host operating system.


Project Overview

This project focuses on creating and configuring a Kali Linux virtual machine using VirtualBox. The lab was designed to introduce Linux fundamentals, system configuration and virtualisation concepts commonly used in cybersecurity environments. The environment is used for continuous hands-on practice.


Lab Environment

  • Host OS: Windows 11
  • Virtualisation platform: VirtualBox
  • Guest OS: Kali Linux (Graphical Desktop)
  • RAM allocation: 2 GB
  • CPU cores: 2
  • Storage: 25 GB

Implementation Process

  • Enabled hardware virtualisation in system BIOS
  • Configured VirtualBox machine settings for stability
  • Installed Kali Linux using the graphical installer
  • Resolved installation failures caused by low memory allocation
  • Completed GRUB bootloader installation
  • Successfully launched and verified the VM environment

Key Learnings

  • How virtualisation supports cybersecurity workflows
  • Basic Linux system navigation and terminal usage
  • How hardware resources affect virtual machine performance
  • The importance of structured troubleshooting and iteration
